# EmoteMenu

Radial emote wheel for Vintage Story. Press `]` to open, select with mouse, click to emote.

Keybind is configurable in Settings → Controls.

## Installation

Drop both zips into your `Mods` folder:
- `RadialMenu-1.0.7.zip`
- `EmoteMenu-1.0.7.zip`

## Building from Source

Requires .NET 8 SDK.

```bash
# Set your VS install path (build.sh tries to auto-detect)
export VINTAGE_STORY="/path/to/vintagestory"

./build.sh
```
